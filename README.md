# 👟 Global Sports Footwear Analysis

A comprehensive end-to-end data analytics project exploring sales performance, customer behavior, pricing strategies, category trends, geographic performance etc in the global sports footwear industry.

## 📌 Project Overview

This project analyzes a multi-year global sports footwear sales dataset (2018–2026) to uncover actionable business insights and support strategic decision-making.
The dataset is synthetically balanced. So it is difficult to decide what is better because of no clear differences.But i believe even a 0.1% difference makes a
big change in long run.

The analysis focuses on:

* Brand Performance & Market Positioning
* Discount Strategy Effectiveness
* Channel Strategy: Online vs Retail
* Geographic Market Prioritization
* Consumer Segmentation by Income Level
* Product Category Strategy
